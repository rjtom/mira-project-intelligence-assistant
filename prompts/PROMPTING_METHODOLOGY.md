# MIRA Prompting Methodology

MIRA uses a hybrid prompting methodology combining 10 techniques.

## The 10 Techniques

1. System Prompting - Agent identity and constraints
2. Role Prompting - Expert persona assignment
3. Instruction Prompting - Explicit behavioral directives
4. Chain-of-Thought (CoT) - Structured reasoning (Risk Assessor)
5. ReAct - Reason before acting (tool use)
6. Grounding Prompting - Prevents hallucination
7. Few-Shot Prompting - Implicit output format examples
8. Constraint Prompting - Hard limits on behavior
9. Persona Prompting - Consistent MIRA voice
10. Output Structure Prompting - Defines expected format
